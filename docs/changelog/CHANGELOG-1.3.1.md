# AG-SubEditor v1.3.1 - Changelog

## Wypalarka napisów

### Dodano

- **Wypalarka (Subtitle Burner)** - nowa funkcja wypalania napisów na stałe w pliki wideo
- Automatyczne pobieranie FFmpeg - aplikacja sama pobiera FFmpeg gdy jest potrzebny
- Obsługa akceleracji sprzętowej NVENC (NVIDIA GPU)

### Naprawiono

- Poprawne wyświetlanie nazwy aplikacji w powiadomieniach Windows
- Dodano Windows App User Model ID dla prawidłowej identyfikacji w systemie
